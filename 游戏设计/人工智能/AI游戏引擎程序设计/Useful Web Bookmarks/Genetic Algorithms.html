Genetic Algorithms

http://www.fe.dis.titech.ac.jp/~gen/robot/robodemo.html
Website with applet showing a stochastically trained robot arm that learns how to drag itself along.

http://www.erim.org/~vparunak/AAMAS02Evolution.pdf
Paper about pheromone path planning, which uses ant algorithms to help with planning.

 

http://ai-depot.com/CollectiveIntelligence/Ant.html
Article with program on emergent intelligence, simulation of ant colony behavior.

 

http://www.cis.upenn.edu/~hollick/genetic/paper2.html
Nice site with information about genetic programming in c/c++.

 

http://lancet.mit.edu/ga/
GALib, a c++ genetic algorithm library.

 

http://www.generation5.org/content/1999/biles.asp
An article on John Al Biles, who made a computer jazz improviser named GenJam that uses genetic algorithms.

 

http://www.ics.uci.edu/~mvle/ics175a/BackgroundPaper.doc
A number of AI research studies done on PacMan, using different algorithms.

 

http://www.genetic-programming.com/published/eetimes060396.html
An article about a genetic program creating analog circuts.

 

http://www.geneticprogramming.com/
Good general site for genetic work. Some links to other AI fields as well.
