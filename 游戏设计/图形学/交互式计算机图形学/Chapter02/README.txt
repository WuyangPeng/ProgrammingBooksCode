The Chapter 2 folder contains the following c programs:

gasket.c: First example in Chapter 2 using random points to generate
the Sierpinski gasket

gasket2.c: Recursive generation of the Sierpinski gasket with triangles

tetra.c: Three dimensional gasket using recursive subdivision with tetrahedra

contour.c: Marching squares program

honolulu.raw: contains integer height data from the Honolulu area that can be used
with marching squares. The first two elements are the number columns and rows in
the array
