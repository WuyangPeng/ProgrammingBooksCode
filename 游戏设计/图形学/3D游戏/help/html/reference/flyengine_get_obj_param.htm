Objects Reference

flyEngine::get_obj_param

Prototype

int get_obj_param(char *objname,char *param,char *value)

Parameters

Parameter Type Description
objname char * object name
param char * parameter name
value char * string to receive the parameter value

Return Value

Returns 0 if parameter was found and value is set, 1 if object not found and 2 if parameter not found.

Remarks

This function is used to get a string representation of an object's parameter.
