Objects Reference

picture::CreatePicture32

Prototype

int CreatePicture32(int xd,int yd)

Parameters

Parameter Type Description
xd int x size in pixels
yd int y size in pixels

Return Value

Returns TRUE if creation is OK, FALSE on any error.

Remarks

This function allocates the memory required for the a picture with the specified dimensions. The buffer allocated is xc*yd*4 bytes.
