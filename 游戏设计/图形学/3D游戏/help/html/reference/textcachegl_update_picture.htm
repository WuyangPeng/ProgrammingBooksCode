Objects Reference

textcacheGL::update_picture

Prototype

void update_picture(int picnum,int sx,int sy,int bytespixel,unsigned char *buf)

Parameters

Parameter Type Description
picnum int picture index
sx int size in x (power of 2)
sy int size in y (power of 2)
bytespixel int bytes per pixel (3 or 4)
buf unsigned char * picture pixels (sx*sy*bytespixel)

Return Value

None

Remarks

This is the OpenGL implementation for the textcache base class virtual function.

This function is used to update the picture in the picture manager with a new pixel array.
