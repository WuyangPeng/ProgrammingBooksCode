Objects Reference

bezier_patch::illuminate

Prototype

void illuminate(vector& p,float rad,vector& color,int shadows);

Parameters

Parameter Type Description
p vector& light point
rad float light radius
color vector& light color
shadows int shadows flag:
0 - no shadows
1 - shadows from specific object (shadow_obj)
2 - shadows from any object

Return Value

None

Remarks

This function illuminates the patch light_map pixles based on the light position, radius and color.
Shadows can be computed for a specific object or all objects using the shadows flag.
