Objects Reference

static_mesh::ray_intersect_test

Prototype

int ray_intersect_test(vector& ro,vector& rd,float rad)

Parameters

Parameter Type Description
ro vector& ray origin
rd vector& ray direction
rad float collision radius

Return Value

Returns TRUE if an intersection is found, FALSE for no intersection.

Remarks

This function implements the ray intersection test virtual bsp_obect function for the bsp faces included in a leaf node.
It passes the ray_intersect_test call to the same function from its objmesh member variable.
