Objects Reference

mesh::set_numfaces

Prototype

void set_numfaces(int nfaces,int local=0,int keep=0);

Parameters

Parameter Type Description
nfaces int new number of faces
local int allocate local faces
keep int keep old faces

Return Value

None

Remarks

This function is used to resize the faces array of a mesh.

If local is not zero, local_faces will also be allocated.

If keep is not zero, the newly allocated array will contain the data from previous faces array.
If new number of faces is bigger then original face count, all old faces are maintained.
If new number of faces is smaller then original face count, only the first nfaces faces will be kept.
