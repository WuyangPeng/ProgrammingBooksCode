class plane

Definition

class plane
{
  public:
  vector normal;
  float d0;

  inline float distance(vector &v)
  { return vec_dot(normal,v)+d0; }
};

Data Members

Member Type Description
normal vector normalized plane normal
d0 float smallest distance from plane to origin.

Remarks

This class implements a plane is 3D space.
Classes like the bsp_node and face derive from a plane.
