This folder has 4 subfolders.

1. Data - Contains the CT data and the Ultrasound data

2. VolumeRenderCartesian - Contains code for Section 40.2.1

3. VolumeRenderPyramid - Contains code for the first half of 
                         the Section 40.2.2, using CT data

3. VolumeRenderPyramidUS - Contains code for the second half of 
                           the Section 40.2.2, using Ultrasound data

VolumeRenderPyramidUS.avi in the folder VolumeRenderPyramidUS shows
a volume rendering of a fetal face.
