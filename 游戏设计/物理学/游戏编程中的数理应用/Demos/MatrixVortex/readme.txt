The Matrix Vortex is designed to teach familiarity with the various matrix and vector operations in a visually stimulating setting.  All of the math code resides in the MatCalc class which is in MatCalc.cpp and MatCalc.h.  The minimum system requirements for the program are a 700 MHz processor, a graphics card with 8 MB of memory, and 64 MB of RAM.  The interface is fairly intuitive and may be navigated with the left mouse button.  Input is entered using the keyboard and backspace will correct mistake.  While entering a matrix or vector, TAB and SHIFT-TAB may also be used to move through the squares.  ESC will quit the program.
