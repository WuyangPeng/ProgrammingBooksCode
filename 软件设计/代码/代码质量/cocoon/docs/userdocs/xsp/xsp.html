XSP learning map

As I find that there is enough information about XSP available (only not together), I give you a list of pointers in the sequence you should read them.

  1. Get Cocoon up and running. Surf to [webhost]/cocoon/slides/slides?section=4 and following. This gives you a first insight in the XSP ground idea.
  2. Read the XSP Logicsheets page. This tells you how to use and make your own XSP logicsheets and XSP pages.
  3. If you're still hungry for more, read the XSP Internals page. This describes how XSP's are handled internally by Cocoon.
