Resources
Apache Cocoon Web Site

The main place is http://xml.apache.org/cocoon/

The C2 documentation is still (and always will be) under development. There is a lot of useful information there and it goes into a lot more detail than here. Read it and digest it. I am certainly doing that as I go along and quite quickly understanding more of the architecture and concepts. I will seek to distill the main points into this site, but the main place is still the supplied documentation.

You will also find the documentation in your C2 installation under {COCOON_HOME}/docs/

Apache Cocoon Users Mail List

Subscribe to the Cocoon users mail list! This is a great resource for asking and answering questions. For more details, and pointers to other related lists, see the documentation.

You can also access an excellent, searchable, archive of the users mail list at The Aims Group. Use this 'cos lots of things are there! Other list archive information is available in the documentation.

Other Pages
  • David Parry has put together a really good XSP Tutorial which is C1.8 based but has all the basic principles you need for C2.
  • Michael Bierenfeld directed me to a cocoon-users posting that has an example taglib (logicsheet) download. Thanks to Gary Clark for the posting and example.
  • Cyril Cambien has developed more examples at his CHello! (Cocoon Hello!) site.
  • Lajos Moczar has developed some more advanced hints and tips at his flashguides site.
