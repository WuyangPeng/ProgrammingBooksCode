IS2ODBC Demo Page

Multithreading Applications in Win32

This page will demonstrate how you can access a JET database using ODBC by making requests one at a time using another process.

To use this demo, you must have correctly installed IS2ODBC.DLL and DBSERVER.EXE as described in 00README.DOC.

The Default Query sends a request to the server to call the default procedure in IS2ODBC.DLL. This call has no parameters.

The Sort by Title Query includes a parameter that will be included in the SORT section of the SQL query.

Finally, Filter by Category selects only a subset of the available records, filtered by the category of the book.
