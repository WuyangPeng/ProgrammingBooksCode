-------------------------------------------------------------
This package contains the www.debugging-guide.com examples.

The examples accompany the book "The Developer's Guide to Debugging"
Groetker, Holtmann, Keding, Wloka, Springer, Aug. 2008

Included are:
- C/C++ source code 
- Makefiles for gmake+GCC builds
- VC8 (Visual Studio 2005) project files
- VC9 (Visual Studio 2008) project files

Tested on the following platforms:
- Windows XP, Cygwin 1.5, GCC 3.4.4
- Windows XP, VC8 SP1
- Windows XP, VC9
- Linux RHEL 3.0, gcc 3.2.1
- Linux Ubuntu 8.04, gcc 4.2.3
- Centos 5.1, gcc 4.1.2
